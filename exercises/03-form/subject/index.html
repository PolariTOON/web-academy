Exercice 3 - Formulaires

Le troisième exercice consiste à réaliser un formulaire.

Pour cela, voici quelques instructions :

  • Le formulaire enverra une requête POST à l'adresse http://webacademy.elmrini.fr/exo_3.php ;

  • Les champs à envoyer sont les suivants :

    • Un champ de texte nom comportant le nom du démon,

    • Un champ de texte prenom comportant le prénom du démon,

    • Un champ numérique age comportant l'âge du démon. Les restrictions sont : âge compris entre 0 et 1000 ans, seulement par décennie (10, 20…),

    • Un champ de date naissance indiquant la date de naissance du démon,

    • Un champ de sélection unique race comportant la race du démon parmi les choix suivants : troll, incube, succube, djinn,

    • Un champ de sélection multiple indiquant les pouvoirs parmi les choix suivants : vol, telepathie, telekinesie.

Vous pourrez tester votre formulaire en envoyant les données à l'adresse mise plus haut.
